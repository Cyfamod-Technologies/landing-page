Help Center

Frequently Asked Questions

Is this really free?

Yes! 100% open-source under MIT license. No hidden fees, no feature restrictions. Download and use for unlimited schools.

What are the system requirements?

Backend: PHP 8.1+, MySQL 5.7+, Laravel 10. Frontend: Node.js 16+, Next.js 13+. Minimum 2GB RAM recommended.

Can I manage multiple schools?

Absolutely! Multi-tenant architecture supports unlimited schools with isolated data per institution.

How do I get support?

Join our Discord community for peer support. GitHub issues for bugs. Enterprise support packages available.

What payment gateways are supported?

Stripe is integrated. Modular architecture allows adding Paystack, Flutterwave, or any gateway.

Can I customize the platform?

Full source code access means unlimited customization. Fork the GitHub repo and modify as needed.

Is there mobile app support?

Web interface is mobile-responsive. Native iOS/Android apps planned for 2026 roadmap.

Can I contribute to development?

Yes! Fork the repo, create feature branches, submit pull requests. See CONTRIBUTING.md for guidelines.

How do I deploy the platform?

Clone from GitHub, run composer install & npm install, configure .env, migrate database. Full guide in docs.

Do I need a VPS or can I use shared hosting?

VPS recommended for best performance. Shared hosting not suitable due to Laravel requirements and performance needs.
